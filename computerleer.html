Wat is een computer?

Een computer is een apparaat dat snel gegevens verwerkt met behulp van programma’s. Het woord komt van het Engelse "to compute", wat berekenen betekent. De resultaten kunnen worden opgeslagen voor later gebruik. Een computer bestaat uit twee hoofdonderdelen: hardware en software.

HARDWARE

Hardware zijn alle tastbare onderdelen van de computer, zoals:

  • Toetsenbord, muis, beeldscherm, systeemkast (met CPU, moederbord, harde schijf).

  • Opslagmedia zoals USB-sticks, geheugenkaarten, SSD’s en harde schijven. Hiermee sla je gegevens op.

    • Geheugen (Memory) is waar de computer informatie tijdelijk of permanent bewaart. Er zijn twee belangrijke soorten geheugen:

  • 1. RAM (Random Access Memory):
    • Tijdelijk geheugen (ook wel werkgeheugen).
    • Verliest alle gegevens zodra de computer uit gemaakt wordt.
    • Kan gelezen en beschreven worden.
    • Wordt gebruikt tijdens het werken (bijv. een open document).
  • 2. ROM (Read Only Memory):
    • Vast geheugen, blijft bewaard ook als de computer uit gemaakt wordt.
    • Bevat gegevens die nodig zijn om de computer op te starten.
    • Kan alleen gelezen worden.


    Geheugen kan permanent of vluchtig zijn:
    • Permanente geheugen (ROM) blijft bewaard, zelfs als de computer uit staat.
    • Vluchtig geheugen (RAM) verliest zijn gegevens zodra de stroom uitvalt.

    Opslagcapaciteit geeft aan hoeveel informatie een opslagapparaat kan bewaren. Dit wordt uitgedrukt in bytes.
    • 1 kilobyte (KB) = 1024 bytes
    • 1 megabyte (MB) = 1024 KB
    • 1 gigabyte (GB) = 1024 MB
    • 1 terabyte (TB) = 1024 GB

    Voorbeelden van opslagmedia:
    • Floppy
    • zipdisk
    • usb-sticks
    • externe harde schijf
    • compact disc
    • Tape media
    • DVD
    • geheugenkaarten

    SOFTWARE

    Software zijn de ontastbare onderdelen, zoals programma’s en instructies. Zonder software werkt een computer niet. Er zijn verschillende soorten software:

    • Applicatiesoftware: Programmas om speciefieke taken te doen (zoals Word, Excel, spelletjes).
    • Besturingssystemen: Programma dat de computer laat werken. Het zorgt ervoor dat je met de computer en andere programmas kan werken (zoals Windows, macOS, Linux).
    • Netwerksoftware (om computers met elkaar te laten communiceren).
    • Drivers (programma’s die hardware goed laten functioneren, zoals een printerdriver).

    Besturingssyteem verder uitgelegd

    • Een besturingssysteem is het belangrijkste programma op een computer.
    • Het zorgt ervoor dat jij met de computer kunt werken.
    • Het regelt alles tussen de knoppen, het scherm en de programma’s; het is als het ‘brein’ van de computer.
    • Zonder besturingssysteem kan een computer niets doen.

    Voorbeelden van Besturingssystemen

    • Windows gemaakt door microsoft
    • macOS gemaakt door apple
    • Linux gemaakt door Linux Torvalds
    • Unix gemaakt door Bell Labs
    • DOS gemaakt door microsoft
